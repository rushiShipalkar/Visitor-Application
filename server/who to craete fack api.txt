                                                            who to create fake API


*******If you want to create a fake (mock) API on your laptop that returns JSON data, there are a few easy methods. The most popular one is using JSON Server with Node.js. It takes only a few minutes.*****

===========================================================================================================================================


Method 1: Using JSON Server (Easiest)
1. Install Node.js

Download and install Node.js from
https://nodejs.org

Check installation:

node -v
npm -v
2. Install JSON Server

Run in terminal:

npm install -g json-server
3. Create a JSON file

Create a file called db.json

Example:

{
  "users": [
    { "id": 1, "name": "Rahul", "email": "rahul@test.com" },
    { "id": 2, "name": "Priya", "email": "priya@test.com" }
  ],
  "products": [
    { "id": 1, "name": "Laptop", "price": 50000 },
    { "id": 2, "name": "Phone", "price": 20000 }
  ]
}
4. Start the fake API

Run:

json-server --watch db.json --port 4000
5. Your API is ready 🎉

Open in browser:

http://localhost:4000/users
http://localhost:4000/products

Example response:

[
  {
    "id": 1,
    "name": "Rahul",
    "email": "rahul@test.com"
  }
]
API Methods Supported
Method	Example
GET	/users
GET by ID	/users/1
POST	add new user
PUT	update user
DELETE	remove user

who to create json file in vs

You can easily create a JSON file in Visual Studio Code by following these steps.

1️⃣ Open Visual Studio Code

Start Visual Studio Code on your laptop.

2️⃣ Open a Folder

Click File

Click Open Folder

Choose or create a project folder.

3️⃣ Create a JSON File

In the left sidebar, click New File 📄

Type the filename:

db.json

Make sure the file ends with .json

4️⃣ Add JSON Data

Now add your JSON data inside the file.

Example:

{
  "users": [
    {
      "id": 1,
      "name": "Rahul",
      "email": "rahul@test.com"
    },
    {
      "id": 2,
      "name": "Priya",
      "email": "priya@test.com"
    }
  ]
}

Save the file using:

Ctrl + S
5️⃣ Check JSON Format

Visual Studio Code automatically highlights JSON syntax and shows errors if formatting is wrong.

Example rules:

Use double quotes " "

Use comma , between objects

Data must be inside { } or [ ]

✅ After creating db.json, you can use it with JSON Server to create a fake API.
